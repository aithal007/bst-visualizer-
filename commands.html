üèóÔ∏è Tree Construction

build_tree([Values], T)
Creates a new binary search tree from a list of values. Each value is inserted in order, maintaining BST properties.
Example: build_tree([50,30,70,20,40,60,80], T)
Creates a tree with root 50, left subtree containing 30,20,40 and right subtree containing 70,60,80
clear
Removes all nodes from the tree, resulting in an empty tree.
Example: clear
Clears the current tree visualization

‚úèÔ∏è Tree Modification

insert(Value, T)
Inserts a new value into the binary search tree. The value is placed in the correct position to maintain BST ordering (left < root < right).
Example: insert(45, T)
Inserts value 45 into the current tree
delete(Value, T)
Removes a value from the tree. Handles three cases: leaf nodes, nodes with one child, and nodes with two children (replaces with in-order successor).
Example: delete(30, T)
Removes node with value 30 from the tree

üîç Tree Search

lookup(Value, T)
Searches for a specific value in the tree. Returns true if found, false otherwise. The visualization shows the search path taken.
Example: lookup(40, T)
Searches for value 40 and highlights the path to it

üö∂ Tree Traversals

inorder(T, L)
Performs an in-order traversal (Left ‚Üí Root ‚Üí Right). For a BST, this produces values in sorted ascending order.
Example: inorder(T, L)
Returns sorted list: [20, 30, 40, 50, 60, 70, 80]
preorder(T, L)
Performs a pre-order traversal (Root ‚Üí Left ‚Üí Right). Useful for creating a copy of the tree or getting prefix notation.
Example: preorder(T, L)
Returns list: [50, 30, 20, 40, 70, 60, 80]
postorder(T, L)
Performs a post-order traversal (Left ‚Üí Right ‚Üí Root). Useful for deleting the tree or getting postfix notation.
Example: postorder(T, L)
Returns list: [20, 40, 30, 60, 80, 70, 50]

üìä Tree Properties

size(T)
Returns the total number of nodes in the tree.
Example: size(T)
Returns: Size = 7
height(T)
Returns the height of the tree (longest path from root to leaf). An empty tree has height 0, single node has height 1.
Example: height(T)
Returns: Height = 3
find_min(T)
Finds and returns the minimum value in the tree (leftmost node).
Example: find_min(T)
Returns: Minimum = 20
find_max(T)
Finds and returns the maximum value in the tree (rightmost node).
Example: find_max(T)
Returns: Maximum = 80
count_leaves(T)
Counts and returns the number of leaf nodes (nodes with no children).
Example: count_leaves(T)
Returns: Leaf count = 4

‚úÖ Tree Validation

is_valid_bst(T)
Validates whether the tree satisfies Binary Search Tree properties (all left descendants < node < all right descendants).
Example: is_valid_bst(T)
Returns: Valid BST or Not a valid BST
